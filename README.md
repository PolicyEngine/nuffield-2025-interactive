# PolicyEngine UK - Nuffield RDA 2025 Interactive

Interactive web application showcasing PolicyEngine's Nuffield Foundation RDA 2025 grant application.

**Live site:** https://policyengine.github.io/nuffield-2025-interactive/

## Features

- **Project summary** with key stats (146 interactive ideas, 3 years, £300k)
- **Searchable/filterable list** of 146 policy interactive ideas across 11 categories
- **Embedded PolicyEngine interactives** (household calculator and policy analysis)
- **PolicyEngine design system** (Inter font, teal primary color)

## Development

```bash
# Install dependencies
bun install

# Run development server
bun dev

# Run tests
bun test

# Build for production
bun run build
```

## Technology stack

- React 19 + TypeScript
- Vite 7
- Vitest for testing
- Bun package manager
- GitHub Pages deployment
